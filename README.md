** Intelligent Resume Analyzer**
** About the Website**
Intelligent Resume Analyzer is an AI-powered web application built using Next.js, Firebase, and Google Gemini AI (via Genkit). 
The platform allows users to upload one or multiple resumes, compare them with a job description, and receive intelligent insights 
such as match score, detailed feedback, strengths, improvement suggestions, and hiring recommendations.
The system is designed for both recruiters and job seekers to make resume screening faster, smarter, and more efficient.
